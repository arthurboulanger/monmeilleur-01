== Description ==

A plugin to allow your website visitors to invite their Facebook friends. It helps increase social media presence and engagement on the website. You can use this plugin as a Widget and can also show the Invite button via a Notification bar.

= Features =

*   Invite Facebook friends with a single click.
*   Show the invite link using hard to ignore notification bar.
*	Can also be used as a widget in the sidebar.
*	Get more exposure on the world's most popular social network.
*   Helps you increase website traffic.
*	Extensive options panel allowing complete control over the plugin.

This plugin is tested upto WordPress 4.0.1

== Installation ==

To install this plugin, please follow the instructions below.

1. Upload `facebook-friend-inviter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'FB Friend Inviter' under 'WordPress Settings' panel.
4. Configure the plugin and start enjoying it.
